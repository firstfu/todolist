# TodoList 應用程式圖示設計

## 設計概述

為 Electron + Vue 3 待辦事項應用設計主要應用程式圖示。

## 設計決策

| 項目 | 決定 |
|------|------|
| 用途 | 應用程式圖示（Dock、任務欄、桌面捷徑） |
| 風格 | 質感立體（macOS Big Sur 風格） |
| 核心圖案 | 勾選符號 ✓ |
| 配色 | 深底金勾 |

## 設計規格

### 外框
- 圓角方形，512x512px，圓角約 22%
- 背景漸層：`#2a3140`（左上）→ `#1a1f26`（右下）
- 1px 深色描邊增加立體感

### 主體圖案
- 居中粗體勾選符號
- 金黃漸層：`#f7c95a`（亮）→ `#dba520`（暗）
- 筆畫粗細：整體寬度的 12-15%
- 勾選角度：約 45°

### 質感細節
- 筆畫末端圓角
- 勾選上方細微高光線
- 背景極淡雜訊紋理
- 輕微內陰影與底部投影

### 尺寸適配
- 512、256、128、64、32、16px
- 小尺寸簡化細節

### 檔案格式
- `.icns` - macOS
- `.ico` - Windows
- `.png` - 各尺寸透明背景
- `.svg` - 原始向量檔
